## 🧪 功能测试项目

### A. 响应式布局测试

#### 断点测试
- [ ] 320px (iPhone SE 竖屏) - 元素不应重叠
- [ ] 375px (标准手机) - 标准布局
- [ ] 414px (大屏手机) - 优化布局
- [ ] 768px (iPad Mini 横屏) - 平板布局
- [ ] 1024px (iPad 横屏) - 大平板布局

#### 安全区域测试
- [ ] 刘海屏顶部不被遮挡
- [ ] 灵动岛不遮挡关键信息
- [ ] 底部手势条不遮挡按钮
- [ ] 打孔屏摄像头不遮挡UI
- [ ] 圆角屏幕边缘元素不被裁切

#### 横竖屏切换
- [ ] 横屏 → 竖屏: 布局正确切换
- [ ] 竖屏 → 横屏: 布局正确切换
- [ ] 方向锁定功能正常
- [ ] 切换时无闪烁或错位

### B. 触摸交互测试

#### 触摸区域大小
- [ ] 所有按钮 ≥ 44x44pt (WCAG标准)
- [ ] 主要按钮 ≥ 56x56pt (舒适尺寸)
- [ ] 射击按钮 ≥ 72x72pt (大型按钮)
- [ ] 虚拟摇杆 ≥ 80x80pt

#### 触摸响应
- [ ] 点击延迟 < 100ms
- [ ] 无双击缩放干扰
- [ ] 无长按菜单弹出
- [ ] 多点触控支持(摇杆+射击同时)
- [ ] 触摸反馈及时(视觉+振动)

#### 手势识别
- [ ] 虚拟摇杆拖动流畅
- [ ] 按钮点击准确
- [ ] 无误触边缘按钮
- [ ] 滑动不触发系统手势

### C. 性能测试

#### 帧率测试
- [ ] 低端设备: 稳定30fps
- [ ] 中端设备: 稳定60fps
- [ ] 高端设备: 稳定60fps (可选120fps)
- [ ] 激烈战斗时帧率不掉

#### 加载性能
- [ ] 首次渲染 < 1s
- [ ] HUD初始化 < 500ms
- [ ] 状态更新 < 16ms (60fps)
- [ ] 动画流畅无卡顿

#### 内存占用
- [ ] 初始内存 < 100MB
- [ ] 运行内存 < 150MB
- [ ] 长时间运行无内存泄漏
- [ ] 页面刷新后内存释放

#### 电池消耗
- [ ] 1小时游戏 < 25%电量
- [ ] 温度控制良好
- [ ] 可调节性能模式

### D. 视觉效果测试

#### 字体可读性
- [ ] 最小字号 ≥ 12px
- [ ] 对比度 ≥ 4.5:1 (WCAG AA)
- [ ] 关键信息对比度 ≥ 7:1 (WCAG AAA)
- [ ] 支持系统字体缩放(100%-200%)

#### 颜色识别
- [ ] 色盲模式测试(红绿色盲)
- [ ] 高对比度模式
- [ ] 状态色清晰区分(安全/警告/危险)
- [ ] 暗光环境可读性

#### 动画流畅度
- [ ] 血量条动画流畅
- [ ] 按钮反馈动画及时
- [ ] 队友状态闪烁清晰
- [ ] 无动画撕裂或跳帧

#### 阴影与发光
- [ ] 文字阴影提高可读性
- [ ] 按钮发光效果醒目
- [ ] 不过度使用影响性能

### E. 可访问性测试

#### 触觉反馈
- [ ] 按钮点击有振动
- [ ] 重要操作振动强度适中
- [ ] 可选关闭振动
- [ ] 不同操作振动模式区分

#### 音频反馈
- [ ] 按钮点击有音效
- [ ] 警告有声音提示
- [ ] 音量可调节
- [ ] 可静音模式

#### 操作辅助
- [ ] 单手操作友好
- [ ] 左右手切换支持
- [ ] 可自定义按钮位置
- [ ] 快捷操作提示
